A simple Gradio app for text-to-speech using Microsoft Edge TTS.

## Setup

\`\`\`\`\`\`bash
pip install -r requirements.txt
\`\`\`\`\`\`

## Run

\`\`\`\`\`\`bash
python app.py
\`\`\`\`\`\`

## Test

\`\`\`\`\`\`bash
python test_edge_tts.py
\`\`\`\`\`\`

## Features

- 300+ voices across 40+ languages
- Adjustable speech rate and pitch
- MP3 audio output

python test_edge_tts.py
